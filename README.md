# Twitter sentiment analysis sample

This is a sample demo showcasing how to set up the credentials for Azure Text Analytics to use sentiment analysis on tweets about Python

## Prerequisites
- [Azure Account](http://aka.ms/azurecredit)
- [Azure Text Analytics](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesTextAnalytics)
- [Twitter Developer Account](https://developer.twitter.com/)
- [Azure Text Analytics package](https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/textanalytics/azure-ai-textanalytics)
- [Azure Identity package](https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/identity/azure-identity)
- [python-twitter package](https://python-twitter.readthedocs.io/en/latest/)
